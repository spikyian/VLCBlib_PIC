Detailed Description

This file provides access to all of the time management functions.

Copyright
Copyright © 2007-2010 Microchip Technology Inc. All rights reserved.

Uses a hardware timer to implement timing functions.

Uses 16bit PIC Timer0. Extends this to 32bit using timerExtension which is incremented on timer0 overflow. Times are stored as TickValue.

Typedef Documentation

◆ TickValue

typedef union _TickValue TickValue

Time unit defined based on IEEE 802.15.4 specification. One tick is equal to one symbol time, or 16us. The Tick structure is four bytes in length and is capable of represent time up to about 19 hours.

Interrupts are required to be enabled in order to extend the timer to 4 bytes in PIC18. PIC24/dsPIC version do not enable or require interrupts.

Gets the current tick counter indicating time since power on.

Returns
the value of the timer

Return the current tick time. PIC18 only: the timer interrupt is disabled for several instruction cycles while the timer value is grabbed. This is to prevent a rollover from incrementing the timer extenders during the read of their values.
