Detailed Description

Baseline functionality required by VLCB and entry points into the application.

Provides the functionality for main() and interrupt processing. Also contains the functionality to handle the list of services. Contains the PIC CONFIG settings.

◆ main()

void main ( void  )

This is the VLCB application start. Do some initialisation, call the factoryReset() if necessary, call the powerUp() service routines. Then call the user's setup() before entering the main loop which calls the user's loop() and service poll().
