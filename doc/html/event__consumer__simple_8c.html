Detailed Description

Implementation of the VLCB Event Consumer service.

Copyright

Author
Ian Hogg
Date
May 2024

The service definition object is called eventConsumerService. Process consumed events. Process Long and Short events. Also handles events with data bytes if HANDLE_DATA_EVENTS is defined. The data is ignored.
