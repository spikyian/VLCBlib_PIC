Implementation of the VLCB EventAcknowledge service.

This service will send a ENACK message if the module has been taught to consume the received event. The module must be in EVENT ACK mode. The service definition object is called eventAckService.

Dependencies on other Services

The Event Acknowledge service depends upon the Event Consumer service. The Event Consumer service MUST be included if the Event Acknowledge Service is included by the module.

The service descriptor for the Event Acknowledge service. The application must include this descriptor within the const Service * const services[] array and include the necessary settings within module.h in order to make use of the PIC event acknowledgement service.
