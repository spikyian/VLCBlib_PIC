Detailed Description

Implementation of the VLCB Event Consumer service.

Copyright
Creative Commons Attribution-NonCommercial-ShareAlike 4.0 International License.
Author
Ian Hogg
Date
Dec 2022

The service definition object is called eventConsumerService. Process consumed events. Process Long and Short events. Also handles events with data bytes if HANDLE_DATA_EVENTS is defined. The data is ignored. If CONSUMER_EVS_AS_ACTIONS is defined then EVs after the Happening are treated as Actions and are added to an Action queue to be processed by the application.
