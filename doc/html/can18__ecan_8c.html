Detailed Description

Implementation of the VLCB CAN Service for the PIC ECAN peripheral.

Copyright
Creative Commons Attribution-NonCommercial-ShareAlike 4.0 International License.
Author
Ian Hogg
Date
Dec 2022

Uses Controller Area Network to carry VLCB messages. This implementation works with the PIC18 ECAN. The service definition object is called canService. The transport interface is called canTransport. Performs self enumeration and CANID collision detection and re-enumeration. Performs loopback of events for Consumes Own Event behaviour. Collects diagnostic data to aid communications fault finding.
