Implementation of the VLCB Event Consumer service.

The service definition object is called eventConsumerService. Process consumed events. Process Long and Short events. Also handles events with data bytes if HANDLE_DATA_EVENTS is defined. The data is ignored. If COMSUMER_EVS_AS_ACTIONS is defined then EVs after the Happening are treated as Actions and are added to an Action queue to be processed by the application.
