LED handling for 2 LEDs.

This file supports modules with two status LEDs. These are normally green and yellow. The green LED may be labelled SLiM and the yellow LED labelled FLiM. Each LED has a state here to indicate if it is on/off/flashing etc.

This looks a bit like a service but not exposed externally.
